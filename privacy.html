Overview

UltraWide Pro ("the Extension") is a Chrome browser extension that applies CSS transforms to HTML5 video elements. This policy explains what data the Extension does and does not collect.

The short version: the Extension collects no personal data whatsoever.

What We Do Not Collect

  • No browsing history or URLs you visit
  • No video content, titles, or metadata
  • No personal identifiers (name, email, IP address, device ID)
  • No usage analytics, events, or telemetry
  • No crash reports sent to any server

What the Extension Does Store

Your preferences (active mode, zoom level, fullscreen-only toggle, and which modes are in the cycle) are stored using chrome.storage.sync. This is Chrome's built-in storage API and syncs across your signed-in Chrome devices. No data is sent to any server operated by the developer.

You can clear all stored preferences at any time by removing and reinstalling the extension, or by using Chrome's "Clear browsing data" feature to clear extension storage.

Permissions Explained

  • storage — to save your mode and zoom preferences locally
  • activeTab — to apply transforms to the video on the currently active tab
  • scripting — to relay keyboard commands from the background worker to the content script
  • host_permissions (<all_urls>) — required to inject the content script on any website that may contain a video player

These permissions are used solely for the Extension's core functionality. None of them are used to collect or transmit data.

Third Parties

The Extension does not use any third-party SDKs, analytics libraries, advertising networks, or remote APIs. There are no trackers, no fonts loaded from external CDNs at runtime, and no outbound network requests from the extension itself.

Children's Privacy

The Extension does not target or knowingly interact with children. No personal data from any user, regardless of age, is collected.

Changes to This Policy

If this policy changes materially, the effective date above will be updated and a new version will be noted in the Release Notes.
